# プライバシーポリシー / Privacy Policy

最終更新日: 2025年1月17日

## 概要

「Webflow管理画面日本語化」（以下、本拡張機能）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

## 収集するデータ

本拡張機能は以下のデータを**ローカルに**保存します：

- **設定データ**: デバッグモードのON/OFF設定
- **翻訳統計**: どの用語が何回翻訳されたかの統計情報

## 収集しないデータ

本拡張機能は以下のデータを**収集しません**：

- 個人を特定できる情報（名前、メールアドレス等）
- 閲覧履歴
- 認証情報（パスワード、トークン等）
- Webflowプロジェクトの内容

## データの送信

本拡張機能は外部サーバーへデータを送信しません。すべての処理はユーザーのブラウザ内でローカルに行われます。

## 第三者への提供

本拡張機能はユーザーデータを第三者に販売、転送、共有しません。

## お問い合わせ

本プライバシーポリシーに関するご質問は、GitHubリポジトリのIssueにてお問い合わせください。

https://github.com/daisuke-ignite/webflow-ja-extension/issues

# Privacy Policy (English)

Last updated: January 17, 2025

## Overview

"Webflow Admin Japanese Localization" (this extension) respects user privacy and is committed to protecting personal information.

## Data We Store

This extension stores the following data **locally**:

- **Settings**: Debug mode ON/OFF preference
- **Translation statistics**: Statistics on which terms were translated and how many times

## Data We Do NOT Collect

This extension does **NOT** collect:

- Personally identifiable information (name, email, etc.)
- Browsing history
- Authentication information (passwords, tokens, etc.)
- Webflow project content

## Data Transmission

This extension does NOT transmit any data to external servers. All processing is done locally within the user's browser.

## Third-Party Sharing

This extension does NOT sell, transfer, or share user data with third parties.

## Contact

For questions about this privacy policy, please open an issue on the GitHub repository.

https://github.com/daisuke-ignite/webflow-ja-extension/issues
